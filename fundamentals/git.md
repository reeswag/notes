# Git

## Basics

```
git clone .... # use to clone an existing repository
git commit -m 'Summarise commit'
git branch # view current brach
git branch like # creates a new brach 'like'
git checkout like # switches to the branch 'like'
git checkout master
git merge like # merges the branched changes to the master branch
git branch -d like # deletes the branch
```
